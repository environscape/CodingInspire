## Modulus8+4 双音序器模块 用户手册

<center>软件版本 v1.0 文档更新日期 2026-07-22</center>

## 一、产品概述

Modulus8+4 是一款 Eurorack 格式双轨音序器模块，集成了两条独立的音序轨道（轨道1 含 8 个步进，轨道2 含 4 个步进），支持四种工作模式、音阶量化、拼接/长度控制以及矩阵导航模式。模块基于 Pico RP2040 处理器实现，提供高精度模拟输出和 4×3 LED 矩阵状态显示。

### 核心特性

| 特性 | 说明 |
|------|------|
| 音序器 | 轨道1 八步 + 轨道2 四步，共 12 个独立 CV 值 |
| 工作模式 | 3 种模式（步进/方向/矩阵） |
| 量化 | 12 种音阶 + 12 个根音可选 |
| 拼接模式 | 轨道1 与轨道2 可拼接为 12 步连续音序 |
| 矩阵模式 | 将轨道1+轨道2 映射为 4×3 矩阵，支持上下左右导航 |
| 输出精度 | 高精度模拟输出，覆盖约 6.6 个八度 |
| 状态显示 | 4×3 LED 矩阵实时指示当前步进与模式 |

## 二、模块操作与接口详解

### 1. 旋钮

模块共有 15 个旋钮，用于设置音序参数和量化选项：

#### 轨道1 旋钮（8 个）

控制轨道1 八个步进的 CV 值。顺时针旋转升高输出电平（约 0–6.6V），对应约 6.6 个八度的音高范围。

**随机触发**：将旋钮拧至最大值附近时，该步进触发将输出随机值。若同时开启了音阶量化，随机值将限制在所选音阶的有效音符范围内。

#### 轨道2 旋钮（4 个）

控制轨道2 四个步进的 CV 值，行为与轨道1 相同。

#### Length/Splice 旋钮

- **Length 模式**（开关抬起）：控制轨道1 的有效步长（1–8 步），超出部分被忽略。
- **Splice 模式**（开关按下）：控制轨道1 与轨道2 的拼接切割点（1–11），将两个音序合并为一条 12 步音序，其中轨道1 占用前 N 步，轨道2 占用后 (12−N) 步。

#### Root 旋钮

选择量化音阶的根音，共 12 个选项（C、C#、D、D#、E、F、F#、G、G#、A、A#、B）。

#### Scale 旋钮

选择量化音阶，共 12 个选项：

| 档位 | 音阶名称 | 说明 |
|------|----------|------|
| 0 | Bypass | 无量化，直接输出旋钮值 |
| 1 | Ionian | 自然大调 |
| 2 | Dorian | 多利亚 |
| 3 | Lydian | 莉迪亚 |
| 4 | Phrygian | 弗里几亚 |
| 5 | Mixolydian | 混合莉迪亚 |
| 6 | Aeolian | 自然小调 |
| 7 | Locrian | 洛克里亚 |
| 8 | Harm.Min | 和声小调 |
| 9 | Pent.Maj | 大调五声音 |
| 10 | Pent.Min | 小调五声音 |
| 11 | SuperLoc | 超级洛克里亚 |

#### Tune 旋钮

微调全局调谐倍率，范围 0.800–1.200 倍，用于校准模块与其他 Eurorack 设备的音高对齐。

### 2. Gate 触发输入接口

四个门控输入接口，用于触发音序器步进。输入触发信号时执行相应操作，具体功能取决于当前工作模式（见第三章）。

- **触发输入1**：轨道1 第一功能（步进前进 / 矩阵下移）
- **触发输入2**：轨道1 第二功能（步进复位 / 反转方向 / 矩阵右移）
- **触发输入3**：轨道2 第一功能（步进前进 / 矩阵上移）
- **触发输入4**：轨道2 第二功能（步进复位 / 反转方向 / 矩阵左移）

### 3. CV 输出接口

两个模拟输出接口，输出范围约 0–6.6V，对应 1V/Oct 标准的约 6.6 个八度。

- **输出1**：轨道1 输出
- **输出2**：轨道2 输出（在矩阵模式下，两路输出保持一致）

### 4. Mode 按钮

每按一次，工作模式在步进模式→方向模式→矩阵模式→步进模式之间循环切换。对应LED会亮起用以提示当前模式。

### 5. Length / Splice 拨动开关

- **按下（Splice）**：Splice 拼接模式，轨道1 与轨道2 拼接为一条 12 步音序。
- **抬起（Length）**：Length 长度模式，轨道1 步长 1–8，轨道2 固定 4 步独立运行。

### 6. LED 矩阵指示

4×3 LED 矩阵实时显示当前模块状态：

- **轨道1 步进指示**：对应轨道1 八个步进，当前活跃步进对应的 LED 亮起。
- **轨道2 步进指示**：对应轨道2 四个步进，当前活跃步进对应的 LED 亮起。
- **模式指示**：分别对应模式 0–2，当前模式的 LED 亮起。
- **矩阵模式**：当前矩阵位置的对应 LED 亮起。

## 三、工作模式详解

### 模式 0：步进模式

基础步进模式，适用于常规音序器使用。

| 触发输入 | 功能 |
|----------|------|
| 触发输入1 | 轨道1 步进前进一步 |
| 触发输入2 | 轨道1 步进复位至第 0 步；若 Splice 模式开启，轨道2 也同步复位 |
| 触发输入3 | 轨道2 步进前进一步 |
| 触发输入4 | 轨道2 步进复位至第 0 步 |

**Splice 模式下的行为**：轨道1 与轨道2 拼接为 12 步连续音序，轨道1 触发超出自身长度时自动借调轨道1 末尾或轨道2 步进，轨道2 触发则在拼接后的后半段范围内循环。

**Length 模式下的行为**：轨道1 按 Length 旋钮值限制步长（1–8），轨道2 固定 4 步独立运行。

### 模式 1：方向模式

在步进基础上增加方向反转功能，支持音序正向/反向播放。

| 触发输入 | 功能 |
|----------|------|
| 触发输入1 | 轨道1 按当前方向前进一步（支持正向/反向） |
| 触发输入2 | 反转轨道1 步进方向 |
| 触发输入3 | 轨道2 按当前方向前进一步（支持正向/反向） |
| 触发输入4 | 反转轨道2 步进方向 |

方向状态在切换模式时保留，切换回模式 0 后方向控制不再生效。

### 模式 2：矩阵模式

将轨道1 的 8 个步进与轨道2 的 4 个步进映射为一个 **4 行 × 3 列** 的矩阵，通过四个触发接口在矩阵中上下左右移动。

#### 矩阵布局

```
           列0          列1          列2
          (轨道1前半)   (轨道1后半)   (轨道2)
     行0   轨道1[0]    轨道1[4]    轨道2[0]
     行1   轨道1[1]    轨道1[5]    轨道2[1]
     行2   轨道1[2]    轨道1[6]    轨道2[2]
     行3   轨道1[3]    轨道1[7]    轨道2[3]
```

#### 导航方式

| 触发输入 | 方向 | 行为说明 |
|----------|------|----------|
| 触发输入1 | ↓ 下移 | 行号 +1，超出底部回到顶部；列号不变 |
| 触发输入2 | → 右移 | 列号 +1，超出最右列回到最左列；行号不变 |
| 触发输入3 | ↑ 上移 | 行号 −1，超出顶部回到底部；列号不变 |
| 触发输入4 | ← 左移 | 列号 −1，超出最左列回到最右列；行号不变 |

#### 输出特性

- 当处于 **列0** 或 **列1**（轨道1 区域）时：输出1 输出对应轨道1 步进的 CV 值，输出2 输出当前行的轨道2 对应值。
- 当处于 **列2**（轨道2 区域）时：输出1 与输出2 输出相同的轨道2 步进 CV 值，两路同步。
- 旋钮拧至最大值附近时同样触发随机值输出。

#### 注意事项

- 进入矩阵模式时，矩阵位置自动复位至左上角（行0、列0）。
- Length/Splice 旋钮在矩阵模式下无效。

## 四、量化系统

### 量化原理

模块内置高精度音阶量化功能，自动将旋钮值吸附到最接近的有效音符上，保证音高准确性。

**量化精度**：每个半音对应约 52 个输出单位，覆盖 C0–C6 共 79 个半音。

**非线性校正**：出厂已内置校正偏移表，确保输出精度。

### 随机触发机制

将任意步进旋钮拧至接近最大值时，该步进触发将输出随机值：

- **未开启量化**：输出 0–4000 范围内的随机值。
- **开启量化**：在所选音阶的所有有效音符中随机选择一个输出。

### 调谐校准

Tune 旋钮提供全局倍率校准（0.800–1.200），可将模块的 1V/Oct 输出与其他 Eurorack 设备对齐。建议在模块初始接入系统时，使用调谐旋钮校准至目标音高。
